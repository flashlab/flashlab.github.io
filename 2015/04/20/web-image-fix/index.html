网络图片处理工具集

发表于 2015-04-20   |   分类于 原创

例行扯淡

写博客，文字丑就需要多多放图来掩盖。一张图片，如果只是存在硬盘里或者用来打印，我们只关注内容就好，但如果准备发布到网上，图片大小和尺寸同样很重要，尤其是独立博客。

进入正题

一般我们的图片来源有三个：拍照、截图、网络下载。动图在这里不做讨论（gif跟flash一样让人抓狂）这里记录几个windows里方便用来web化图片的小程序。

PhotoScaler

来自xylemstudios的小工具，需要.net支持，用来批量缩小图片大小，话说他家的ColorUtility也很不错。缩小体积最见效的就是减像素，最简单的办法就是用自带的画图，想批量的话就要靠这个小玩意了。你可以按百分比，也可以按像素，一般同一文章中的图片调整到同样大小比较好。另外可以选择压缩算法，一般选balance。当然，旋转，翻转也是支持的。缺点也有，不支持预览，不能裁剪也算吧。个人想法，千万像素的PP不适合出现在普通博文里，除非你是在秀。

color quantizer

非常强大的PNG优化/压缩工具，大部分参数都看不懂orz，但是提供了一键批量转换，真的很贴心啊。缺点嘛，大概就是压缩过程实在太慢。orz

pngquant

主页
经过一段时间尝试，感觉这个是最好用的，没有界面，直接拖放操作，压缩率也很感人，重要的是稳定，不会像color quantizer偶尔会处理失败。

Caesium

主页
尝试的机会不多，留着备用。

packJPG

老牌jpg无损压缩工具，压缩率在20%左右。核心是命令行程序，有个界面版但好像不是最新的。当然，也可以下载合辑packARC，作者似乎是想pack所有能pack的media文件。但是后来发现这个压缩完就不是jpg格式了。orz

antrenamer2

批量重命名。嗯，你会需要的。

Photoshop

鉴于我只会用索套扣扣劣质图，所以也算小工具吧！

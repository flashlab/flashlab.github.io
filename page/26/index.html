北风吹，雪花飘

发表于 2010-01-05   |   分类于 原创

两个小时前，武汉这个冬天迎来了第二场大雪

漫天的雪花充溢着整个空间，像雨点一样坠落到水泥路面，瞬间就消失的无影无踪。远远望去，狮子山已经白茫茫的一片了，真有点“山舞银蛇，原驰蜡象的”的感觉。

还有两个星期就到寒假。有企盼也有些许不安。走到一零年代，年味儿是越来越淡，过时过节连形式都省了不少，小时候四代同堂，煮饺子，放鞭炮的乐趣也只能停留在记忆里了。年初我家从山坳坳的小平房搬到了镇上的商品楼，表哥表姐都见不着面，整个春节，几乎都是我一个人我在家里自娱自乐，实在无趣。所以我不安，怕又要失去理想中的那份热闹劲儿。有时我甚至盼着年兽复活一次，因为现代人仿佛已经把它忽视了。

搭桥游戏

发表于 2010-01-03   |   分类于 分享

“看似简单”的物理游戏

daqiao.swf

游戏的目的就是在沟壑的地方搭桥，使用的材料有木板、木梁、钢板、钢梁，不同材料的性质和价格不同，你要做的就是用最少的美元搭建能让小人搬回“货物“的桥。这个游戏有全球排行（highscores），打完之后你可以看下自己处于什么水平！

元旦记事

发表于 2010-01-03   |   分类于 原创

阿嚏～天儿是越来越冷，再过几日就到小寒了。
今年武汉有些阴冷潮湿，新年第一天就下起了连阴雨。出校和几个哥们儿游荡一整天，晚上回来手脚都几乎没了知觉，十根手指冻得像猪蹄儿一样。唉，领教过江城冬天就要念叨夏天的好了。不是有句谚语“小寒大寒寒得透，来年春天天暖和”，2010应该会风调雨顺吧！
